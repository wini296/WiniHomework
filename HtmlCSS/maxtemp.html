Max Temperature



This project`s aim was to analyze the weather when we approach to the equator.


To make a this analysis happen we have colleced data from the OpenWeatherMap API about 500+ cities across the globe randomly.


We compare plots latitude vs temperature, humidity, cloudiness, windspeed. We can conclude that latitude effects none of these variables but the temperature.
